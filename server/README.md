# Artvalue - Backend 

## Running the project
`npm run backserver`

## User API
### GET /users 
Returns all the users

### GET /users/wid 
Returns user detail with specific walletID

### POST /users 
Create user with given values
"walletid", "userName", "email", "personType", "newsletter"


### PUT /users 
Update user with values
